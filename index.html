Introduction

What is Horus ?

Horus is just a micro php framework helps you tos start your next web application .
Horus 4 doesn't like to has any complex code.
Using this version of horus 4.x.x you can build you own distribution on top of horus .
Horus 4 Comes with:
  • SQL Class Based on PDO
  • SQL-Table Class, For CRUD
  • Events/Hooks
  • Simple universal class loader
  • Internal HTTP organizer Response manager
  • Smart light uri router
Horus 4 is ready to work under any design pattern such as MVC/HMVC and it's router is ready for internal requests Sub Requests .

Hello World

After downloading horus, and extracting it in your server, just open index.php
you will see something like this: 


    // load horus kernel
    require_once 'Horus/Horus.php';

    // sart it
    $app = new Horus;

    // print hello world
    echo 'Hello world';

    //run it 
    $app->run();
That is the simplest way to use horus .
But there is no routing ?!!! i need it !!
Ok, don't worry , you can just enable routing :D .


    // load horus kernel
    require_once 'Horus/Horus.php';

    // sart it
    // and tell horus to use router !
    // know more about this configs in #Configurations section .
    $app = new Horus(array( 'horus.use_router' => true ));

    // print hello world 
    // only when the index page is requested
    // through any request method (GET, POST, PUT, ... etc) .
    $app->router->any('/', function(){
        echo "Hello world";
    });

    //run it 
    $app->run();
Now close this dialog and see horus #Configuraions dialog .

Configurations

Horus 4 nearly has zero configurations , just mini to do better .

How to config ?

Ok, there is 2 way:
  • On construction like what we did with using router
  • After construction, any where in your code
On construction 
 new Horus(array('key' => 'value', 'key2' => 'value2'));
After construction
 
// start horus
$app = new Horus; 
// configs
$app->config(array('key' => 'value', 'key2' => 'value2'));
// to get a config
echo $app->config('key');
// to get all
var_dump($app->config());

Available Configurations

Config name Config description Config value
horus.use_router Do you want horus to load router ? Bool true/false
horus.use_sql Do you want horus to load sql db manager ? Bool true/false
horus.use one line to load SQL/Router '*' will load all (SQL and Router),
array('SQL', 'Router') to load SQL and Router or just choose one to load
horus.error_404 A callback to handle 404 error docs must be a valid Callback
horus.http_version what is your http interface version ? String default 1.1
horus.enable_simulator Simulate url_rewrite ? (if your server doesn't suuport it) Bool true/false
horus.controllers_dir if you want horus to autoload the class that handles a routed uri string

Response

The response manager is a http response manager that organize your application headers and output .
After organizing it sends headers then it sends the output, so there will be no errors like headers already sent .
But to use it, you should know: 


// start horus
$app = new Horus;

// send a header like header()
$app->response->header($header_string, $replace = false, $status = 200);

// set status code
$app->response->status(200);

// get the status code
echo $app->response->status();

// stop the application and send message with status code
$app->response->halt($code, $message = null);

// redirect
$app->response->redirect($to, $http_code = 302);

// cache wep page over http for some time
$app->response->cache($ttl);

// send all headers then the output
// this method by default called in Horus::run() method
// so you don't need to use it .
$app->response->send($output, $return = false); // $return (true) or echo (false) 

// run it
$app->run();

Routing system

Horus Router is a lightweight smart url router that can be used in any application also ready for HMVC / MVC .
It's event driven, you list a callback with it's uri-pattern, so when the uri hit in the browser
the callback will be executed .
Horus Router also supports any http method
Also it supports class routing that helps you make a class to handle a uri with it's childs . 


// start horus
// use the router
$app = new Horus(array( 'horus.use_router' => true ));

// let's handle the root '/' in a get request
$app->router->get('/', function(){
    echo "Hi";
});

// let's handle the root '/' in a post request
$app->router->post('/', function(){
    echo "Hi";
});

// let's handle the root '/' in a put request
$app->router->put('/', function(){
    echo "Hi";
});

// let's handle the root '/' in a xx request
$app->router->xx('/', function(){
    echo "Hi";
});

// let's handle the root '/' in a get and post request
$app->router->get_post('/', function(){
    echo "Hi";
});

// let's handle the root '/' in a get,post and put request
$app->router->get_post_put('/', function(){
    echo "Hi";
});

// let's handle the root '/' in any request
$app->router->any('/', function(){
    echo "Hi";
});

// to handle multiple patterns with one callback
// let's handle the root '/' and 'index' in a get request
$app->router->any(array('/', 'index'), function(){
    echo "Hi";
});

// named params
$app->router->any('/hi/{num}', function($num){
    echo "Hi $num";
});

// named params
$app->router->any('/hi/{num}/{alpha}', function($num, $alpha){
    echo "Hi $num and $alpha";
});

// handle a path with it's all childs
$app->router->any('/hi/{*}', function(){
    echo "Hi i'll be here for all";
});

// nested routes
$app->router->any('/hi/{*}', function(){
    echo "Hi i'll be here for all";
    $app->router->any('/hi/page-1', function(){
        echo "i'll be here only for 'page-1'";
        // and so ...
    }, true); // ---> notecie the 3rd param ! set to true in this case
});


// all available shortcuts:
/*
'{num}'      =>  '([0-9\.,]+)',
'{alpha}'    =>  '([a-zA-Z]+)',
'{alnum}'    =>  '([a-zA-Z0-9\.]+)',
'{str}'      =>  '([a-zA-Z0-9-_\.]+)',
'{any}'      =>  '(.+)',
'{*}'        =>  '?|(.*?)'
*/

// you can add tour own shortcut
$app->router->shortcut(':my:', '(.*?)');

// you can remove a shortcut
$app->router->unshortcut(':my:');

// do you know that you can call another route from
// any route
// say we have this login router
$app->router->post('/login', function(){
    var_dump($_POST);
});

// say that we have this index router
// and we want to call login router without redirect !
$app->router->any('/', function(){
    // get horus instance
    $app = Horus::instance;
    
    // call login handler over HTTP POST
    // also you can pass array of params to it
    // and show it's response
    echo $app->router->request
    (
        // first param is the request uri
        '/login', 
        // second param is request method
        'POST', 
        // third param is array of request params (optional)
        array('user'=>'mohammed', 'pass'=>'test')
    );
});

// instead of functions/anonymous functions, you can route 
// a full class
// make this class
class test
{
    // must be exists
    function index()
    {
        echo 'hi we are in the index';
    }
    
    function page_1()
    {
        echo "page_1";
    }
    
    function _page()
    {
        // cannot be accessed from the router
        // as the router ignores any methods starts 
        // with '_'  (underscore)
    }
}

// make the 'test class' handles '/' uri
// if you set the 'horus.controllers_dir' 
// you don't need to load the class manually, just
// type its name and will be loaded ( class filename = class name )
$app->router->any('/', 'test');

// now when you call '/' in your browser
// test::index() will be called

// and when you call '/page_1' in your browser
// test::page_1() will be called

// and when you call '/page-1' in your browser
// test::page_1() will be called

// and when you call '/page-1.extension' in your browser
// test::page_1() will be called

// do you know that you can use horus controller
// just extends Horus_Controller class
// then you have full access to Horus object
class test extends Horus_Conroller
{
    // must be exists
    function index()
    {
        // now you could access any thing in Horus class using '$this'
        // example:
        // $this->config(....)
        echo 'hi we are in the index';
    }
}

// do you want to get segments of current uri ?
$segments = $app->router->segments();

// or get only the first segments ?
echo $app->router->segments(0);

// more advanced ?
// ok, say that we are in '/page1/page2/page3'
// and you want to get segments starting from '/page2'
// it uses regex, so you must escape regex chars
// so
$segments = $app->router->segments('', '\/page1\/page2');

// run it
$app->run();

SQL system

Horus comes with its own SQL class that is extends PDO and adds some new methods .


// start horus
$app = new Horus(array( 'horus.use_sql' => true ));

// basic connection ( just like pdo )
$app->sql->connect($dns, $user, $pass, $array_options);

// or connect to any driver of the following
$app->sql->mysql($host, $db, $user, $pass);
$app->sql->mariaDB($host, $db, $user, $pass);
$app->sql->mssql($host, $db, $user, $pass);
$app->sql->oracle($host, $db, $user, $pass);
$app->sql->pgsql($host, $db, $user, $pass);
$app->sql->sqlite($file, $prsistent = false);

// to get the current used driver
echo $app->sql->driver();

// to get the current used database
echo $app->sql->database();

// to do a query
// returns false if not-done or $app->sql object if ok .
$app->sql->query($sql_statment, $inputs_array);
// example:
$app->sql->query('select * from tablename where mycolumn = ? and my2column  = ?', array('value1', value2));

// it comes with some basic shortcut methods
// insert
$app->sql->insert('tablename', array(
    // 1)- array of columns
    array('name', 'email', 'password'),
    
    // 2)- array(s) of inserts ! in just one query !!
    array('myname-1', 'mymail-1', 'mypass-1'),
    array('myname-2', 'mymail-2', 'mypass-2') // more inserts
));

// replace
$app->sql->insert('tablename', array(
    // 1)- array of columns
    array('name', 'email', 'password'),
    
    // 2)- array(s) of inserts ! in just one query !!
    array('myname-1', 'mymail-1', 'mypass-1'),
    array('myname-2', 'mymail-2', 'mypass-2') // more inserts
), true); // 3rd param set to true in replace

// do you need simple insert/replace for single insert ?
$app->sql->insert('tablename', array(
    'name'  =>  'myname',
    'email' =>  'my@em.ail',
    'password' => 'my secret word'
)); 
// that is an insert, but for replace
// but for replace set the last param to true or 1, like mltiple inserts

// update
// update tablename set name = ? , email = ? where id = ?
$app->sql->update('tablename', array(
    'name'  =>  'newname',
    'email' => 'newmail'
), 'id = ?', 5);

// delete
// delete from tablename where id = ?
$app->sql->delete('tablename', 'id = ?', 5);

// set name (charset)
$app->sql->charset('utf8');

// get current database tables array 
$tables = $app->sql->tables();

// or for another db
$tables = $app->sql->tables('mysql');

// if table_exists of current used database ?
$app->sql->table_exists('my_table');

// if table_exists of another used database ?
$app->sql->table_exists('my_table', 'mydatabase');

// you can check if multiple tables exists or not
// just replace 'my_table' with array of tables to check

// get current database table columns array 
$columns = $app->sql->columns('my_table');

// get another database table columns array 
$columns = $app->sql->columns('my_table', 'my_database');

// a column exists (current_database) ?
$app->sql->column_exists('my_table', 'my_column');

// or another database
$app->sql->column_exists('my_table', 'my_column', 'mydatabase');

// you can check if multiple columns exists or not
// just replace 'my_columns' with array of columns to check

# -------------------------------------------------

// Do you know that there is another way and simple ?
// example, work on a teble called 'posts'
// this will create a new full object of Horus_SQL_Table class
$posts = $app->sql_table->using('posts');

// now add entries
$posts->add(array(
    'post_title'    =>  'title here',
    'post_time'     =>  time(),
    'post_content'  =>  'content here'
));

// or multiple entries
$posts->add(array(
    array('post_title',     'post_time',    'post_content'),
    array('title 1',        time(),         'content 1'),
    array('title 2',        time(),         'content 2'),
    array('title 3',        time(),         'content 3')
));

// replace entries (same as add single and multiple support, just change add to rep)
$posts->rep(array(
    'post_title'    =>  'title here',
    'post_time'     =>  time(),
    'post_content'  =>  'content here'
));
$posts->rep(array(
    array('post_title',     'post_time',    'post_content'),
    array('title 1',        time(),         'content 1'),
    array('title 2',        time(),         'content 2'),
    array('title 3',        time(),         'content 3')
));

// delete entries
// $posts->del($where, $inputs);
// $where : without 'WHERE' keyword
$posts->del('post_id = ?',                      5);
$posts->del('post_title = ? and post_id = ?',   array('title',  5));

// edit entries
// $posts->edit(array $edits,   $where, $inputs);
// $where : without 'WHERE' keyword
$posts->edit(array(
    'post_title'    =>  'new title'
),  'post_id = ?',  5);

// fetch one
// $posts->get_one($where, $inputs, $fetch = PDO::FETCH_ASSOC)
// $where : without 'WHERE' keyword
$posts->get_one('post_id = ?',  5);

// fetch all
// $posts->get_all($where = null, $inputs = null, $more_sql = null, $more_inputs = null)
// $where : without 'WHERE' keyword
// $more_sql: if you want to add more sql statement
// $more_inputs: inputs for $more_sql
all = $posts->get_all('post_id = ?',  5)->fetchAll(PDO::FETCH_ASSOC);

// count rows
// $posts->count($where = null, $inputs = null)
// $where : without 'WHERE' keyword
// returns float number
$number = $posts->count('post_title like ?', '%title%');

// optimize
$posts->optimize();

// drop
$posts->drop();

// truncate
$posts->truncate();

// is the table exists
$posts->exists();

// get table columns
$cols = $posts->cols();

// if column exists ?
$posts->exists('my_col');

// or array of columns
$posts->exists(array('col1', 'col2'));

// you can access any $sql method easily
$posts->rowCount();
$posts->lastInsertId();

// run it
$app->run();

Evetns/Hooks

Horus comes with simple events/hooks system to help you extend your application . 

// start horus
$app = new Horus;

// listen listener(s) to an event
$app->events->listen('tag.1', function(){
    echo 'event-1';
});

$app->events->listen('tag.1', function($x){
    echo 'event-'.$x;
});

// disptch / trigger event and pass args
$app->events->trigger('tag.1', array(5));

// do you know that the last argument will contain 
// the value of last executed event !
// this is useful for filters

// run it
$app->run();

Extend Horus 


// start horus
$app = new Horus;

// assign a var
$app->var = 'value';
// get it
echo $app->var;

// assign method
$app->method = function(){
    echo "hi";
};
// call it
$app->method();

// unset
unset($app->var);

// isset
isset($app->var);


// you can also listen events to extend horus
// just use those
// horus.before.dispatch
// horus.after.dispatch
// horus.before.output
// horus.after.output
// the output is stored in $app->__output


// run it
$app->run();
