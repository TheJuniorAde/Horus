Horus Framework

Horus Framework is a simple yet powerful micro php framework perfect for getting programmers start .
No more big size framework or useless libraries just strong code-base and well designed skeleton for any type of app(s)

Features

  • Simple yet powerful router
    • Regex Based and comes with regex shortcuts
    • Any valid callback for routing
    • Class Routing support 'pattern/method/param/param/...etc'
    • RESTful Requests Routing
    • Multi Methods Routing
    • Multi Patterns Per Class or Callback
    • Nested Routes Support from version 1.1.1
  • Full Output Control
    • Extensible, you can assign any event/hook filter and edit it
    • GZIP, you can optionaly enable or disable gzip
    • HTML Tidy, you can optionaly enable or disable html fixer
    • XML Tidy, you can optionaly enable or disable xml fixer
    • Minifier, you can optionaly enable/disable output minifier to remove dublicated spaces,lines,tabs, .. etc
  • Custom views
  • Supports Multi Languages Applications from version 1.2.0
  • HTTP Class from version 1.1.1
  • Good helpers for common functions
  • Better events system (hooks or event listener/dispatcher)
  • Fully extensible
  • Very simple configurations
  • Very simple database class (improves PDO)
  • Assign/Unset any method on the fly
  • Secure Session functions
  • Works on most basic PHP hosting planes
  • Framework Size < 100 kb
